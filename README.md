AquaTrack is an IoT and AI-based water monitoring system designed to reduce water wastage,
enforce daily usage limits, and promote sustainable water management.

## Features
- Real-time water usage monitoring
- IoT-based smart metering
- Cloud data storage
- AI-based water usage prediction
- Alerts for excess usage
- Government & user dashboards

## Technologies Used
- ESP32 / Arduino
- Python (Flask, MQTT)
- Machine Learning (Linear Regression)
- Firebase / SQLite
- MQTT Protocol

## Project Modules
1. Smart Metering
2. Data Communication
3. Cloud Analytics
4. Alerts & Cut-offs
5. Industrial Recycling Monitoring
6. Public Reuse System
